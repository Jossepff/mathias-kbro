1.¿Qué es un Protocolos Http?

Http es un protocolo sin estado, por lo que no guarda ninguna informacion sobre conexiones anteriores

2.¿Qué es un Query String?

Query string o, en español, cadena de consulta es un término informático que se utiliza para hacer referencia a una interacción con una base de datos.

3.¿Qué es un Verbos HTTP?

Un método HTTP, a veces denominado verbo HTTP, indica la acción que la solicitud HTTP espera del servidor consultado.

4.¿Qué es un DNS?

Es el sistema de Nombres de dominio(Domain Name System), que traduce las direcciones IP a normbres sencillo, para que los navegadores puedan cargar las paginas web, programas o documentos de internet que desean visitar.

5.¿Qué es un CDN?

es un conjunto de servidores que contienen copias locales de archivos que se encuentran almacenados originalmente en otros servidores alejados geográficamente. De esta manera, cuando el ordenador de un usuario realiza la petición de un contenido que se halle en un servidor muy lejano, su solicitud es redirigida hacia las copias presentes en el CDN más cercano, lo que reduce considerablemente los tiempos de carga. Además, se evita que los servidores originales colapsen ante una cascada de peticiones.
